# n8n Templates Hub ⚡

Bem-vindo ao meu repositório de automações para **n8n**. Aqui compartilho fluxos prontos para uso, focados em resolver problemas reais com o poder da Inteligência Artificial e automação low-code.

---

## 🤖 Templates Disponíveis

### 1. WhatsApp AI Multimodal Agent
Este é um assistente inteligente para WhatsApp que processa **Texto, Áudio, Imagem e Vídeo** em um único fluxo, utilizando o **Google Gemini 1.5 Pro**.

* **Arquivo:** [`whatsapp-ai-multimodal-agent.json`](./whatsapp-ai-multimodal-agent.json)
* **Destaques:**
    * 🎙️ **Áudio:** Transcrição e análise de notas de voz.
    * 🖼️ **Visão:** Descrição de imagens e extração de texto (OCR).
    * 🎬 **Vídeo:** Resumo visual de conteúdos de vídeo.
    * 🧠 **Memória:** Agente que mantém o contexto da conversa.

---

## 🚀 Como utilizar os templates

1.  **Download:** Escolha o arquivo `.json` desejado na lista acima e faça o download.
2.  **Importação:** No seu n8n, clique em **Workflows** > **Import from File**.
3.  **Configuração:**
    * Configure as credenciais necessárias (ex: WhatsApp API, Google Gemini).
    * Verifique os nós de gatilho (Webhook) para garantir que estão ativos.
4.  **Execução:** Ative o workflow e comece a automatizar!

## 🛠️ Requisitos Gerais
- n8n (v1.0 ou superior recomendado)
- Credenciais de API para os serviços utilizados (verificar descrição de cada template)
